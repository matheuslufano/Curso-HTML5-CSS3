Links para download

(parâmetro: download=" " e type="" )

Os link de dowload vair usar a mesma tag '< a >' para de criar um link, dentro da tag basta digitar o parâmetro download:

parâmetro: download="meulivro.pdf"
Esse parâmetro indica onde estar o arquivo, para isso o arquivo em pdf ou zip precisa estar dentro da pasta/servidor

parâmetro: type=" "
indica qual o formato do arquivo a ser baixado
obs.: cada arquivo tem um type diferente, como:

type="application/zip"
type="application/pdf"

  • Baixar o livro em PDF
  • Livro compactado em ZIP
